About C-Chart

A C-Chart (Count Chart) is a type of control chart used to monitor the number of defects or nonconformities in a process when the sample size is constant. It tracks attribute data - specifically counting the number of defects in each sample.

What it measures: The number of defects, errors, or nonconforming items in fixed-size samples over time.

When to use: When you want to monitor process stability for attributes where the sample size remains constant.
